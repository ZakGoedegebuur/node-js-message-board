# node-js-messenger
Simple node js server which serves and communicates with a frameworkless message board website

# Initialise
After cloning to your computer and running npm install, run "npm run init" to init. Enter the ip address you want the server to run on when prompted, and everything else will be taken care of.

# Running
To run the server on your computer enter "node ." from the root directory and the program will run on the address printed to the console. To go to the served site, copy the address into a browser.

# Known Issues
The system may not work when using safari on an imac.
